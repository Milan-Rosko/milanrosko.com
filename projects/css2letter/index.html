Overview

CSS2Letter is a DIN A4 letter composed directly in HTML. Open composer/index.html in a text editor and change the sender, recipient, reference fields, subject, date, and footer in their existing HTML elements.

Write the letter body inside <main id="body-main">, using one <p>...</p> element for each paragraph. Paragraph classes remain attached when the pagination engine moves or divides text across sheets. Save the HTML, refresh it in the browser, and use Print to produce the PDF.

Trivia: DIN A paper preserves the ratio 1:√2 under halving and rotation.

Compose a Letter

Work in a local copy of composer/index.html. This single HTML file is both the letter template and the editable letter; there is no separate content file.

  1. Replace the text in the address window, information block, subject, date, and footer.
  2. Find <main id="body-main"> and write the letter there. Put each paragraph in its own <p>...</p> element.
  3. Add a class directly to a paragraph when needed, for example <p class="signature-name">...</p>. Pagination retains the class when it moves or splits the paragraph.
  4. Save the HTML and open or refresh it in the browser. Pages are created automatically from the HTML content.
  5. Select Print to produce the finished A4 PDF.
