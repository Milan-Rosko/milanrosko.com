Overview

CSS2Letter is a local DIN A4 text composer. It produces print-ready documents in the browser and exposes typography, spacing, and page geometry as CSS.

Pagination is controlled by paragraph codes. Break, continuation, and spacing rules belong to paragraph roles rather than fixed page positions, so the division into sheets can be recomputed after the text changes.

Project Files

The project consists of a website page and a self-contained composer:

  • index.html presents this overview, navigation, and project references.
  • css2letter.css contains the recursive DIN proportion illustration.
  • composer/index.html defines the DIN 5008 letter template and print command.
  • composer/styles/din5008.css controls A4 geometry, typography, letter fields, screen preview, and PDF print output.
  • composer/scripts/letter-content.js contains the editable sender, recipient, metadata, and paragraph-coded letter content.
  • composer/scripts/pagination.js measures the available page area and moves or divides content across additional sheets.
  • composer/scripts/letter-engine.js renders the content, waits for fonts and mathematics, and starts pagination.
  • composer/assets/logo.png and signature.png provide the visible letterhead artwork.
  • composer/assets/reference/din.png and mr.png are visual references for template development.
